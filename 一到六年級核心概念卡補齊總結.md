
## 維護注意

- 新增或調整核心概念後，先更新 `數學學習地圖/data/concepts.csv`、`activity_concepts.csv`、`relations.csv` 與必要的 `concept_candidates.csv` 狀態。
- 再執行對應學期的 `sync-g*a/b-concepts.js`，將來源 CSV 同步到 `knowledge-map/data/units.json`。
- 最後執行 `node scripts/check-all-completion-gaps.js` 做總回歸。
- 提交 `knowledge-map` repo 時只 stage 明確要部署的檔案，避免把既有備份檔、草稿清單或未追蹤檔一起提交。
