# AGENTS.md — knowledge-map

## 開始前必讀

1. 讀本資料夾 `CLAUDE.md`。
2. 讀本資料夾 `handoff.md`。
3. 遵守上層與全域規則。

## 專案定位

- 國小數學知識地圖與資料品質工具。
- 技術：靜態 HTML/JS + `data/` 資料 + scripts 檢查。
- 部署：GitHub Pages。

## 修改規則

- 修改既有 HTML/JS/資料檔前先建立 `.bak`。
- 資料更新要跑對應檢查腳本。
- 不用推測補概念；資料不足要標示待確認。
- 上層 `三下數學` 只作索引，進入本專案後以本資料夾文件為主。

## 結束時

更新本資料夾固定文件；資料里程碑要更新 `CHANGELOG.md`，踩坑要更新 `PITFALLS.md`。
